# Selectec
College module for matriculation
## Status
Uncompleted

### Technologies

- ReactJs

  - Bootstrap (Botswatch theme `Material`)
  - Context-API

- NodeJs

  - ExpressJs
  - MongoDB
  - JWT

## Installation

Use the next command in root folder and clientapp

```bash
npm install
```
Add your mongodb URL in file 

```bash
config/dev
```

```bash
MONGODB_URI: "your mongodb URL",
```


## Run the project

DEV Environment

```bash
npm run dev
```
